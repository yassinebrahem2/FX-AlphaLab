\# Clean Price Data (Week 4 Definition)



This folder contains \*\*cleaned but non-enriched\*\* OHLC price data.



Cleaning in Week 4 means:

\- UTC timestamps

\- canonical column names

\- schema validation passed

\- metadata attached

\- MT5-specific fields removed



Cleaning does NOT include:

\- indicators

\- resampling

\- aggregation

\- filtering

\- signal generation

\- trend detection



Clean data is suitable for:

\- database ingestion

\- exploratory analysis

\- downstream agent consumption
